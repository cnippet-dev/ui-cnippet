## Installation

Follow these simple steps to add the Sonner component to your project:

<Steps>

<Step>Install the following dependencies:</Step>

```bash
npm install sonner
```

<Step>Copy and paste the following code into your project.</Step>

<CollapsibleSource name="sonner" path="components/ui/sonner.tsx" />

<Step>Add the Sonner component to your project.</Step>

`app/layout.tsx`

```tsx
import { Toaster as Sonner } from "@/components/ui/sonner";

export default function RootLayout({ children }) {
    return (
        <html lang="en">
            <head />
            <body>
                <main>{children}</main>
                <Sonner />
            </body>
        </html>
    );
}
```

<Step>Update the import paths to match your project setup.</Step>

</Steps>
