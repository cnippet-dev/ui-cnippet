## Prerequisites

Before you begin, make sure you have:

- Next.js 15+ and React 19+
- Tailwind CSS configured in your project
- Basic knowledge of React and TypeScript

## Method 1: Full Setup (Recommended)

Get all components at once by cloning our starter repository.

### Step 1: Get the Code

```bash
# Clone the repository
git clone https://github.com/cnippet-dev/all-elements
```

### Step 2: Copy Essential Folders

Copy these folders into your Next.js project:

- `components/` - All UI components
- `hooks/` - Custom React hooks
- `lib/` - Utility functions

### Step 3: Install the following dependencies:


### Step 4: Set Up Theme Provider

To enable theme functionality throughout your application, wrap your app with the ThemeProvider component from `next-themes`.

First, create a new file `providers/theme-provider.tsx` and add the following code:

```tsx
"use client";

import * as React from "react";
import { ThemeProvider as NextThemesProvider } from "next-themes";
import { type ThemeProviderProps } from "next-themes/dist/types";

export default function ThemeProvider({
    children,
    ...props
}: ThemeProviderProps) {
    return <NextThemesProvider {...props}>{children}</NextThemesProvider>;
}
```

Next, update your `app/layout.tsx` file to integrate the ThemeProvider:

```tsx
import ThemeProvider from "@/providers/theme-provider";

export default function RootLayout({
  children,
}: {
  children: React.ReactNode;
}) {
  return (
    <html lang="en" suppressHydrationWarning>
      <body>
        <ThemeProvider
          attribute="class"
          defaultTheme="light"
          enableSystem
          disableTransitionOnChange
        >
          {children}
        </ThemeProvider>
      </body>
    </html>
  );
}
```

### Step 5: Configure Next Cloudinary

Set up Cloudinary to enable media uploads and optimized delivery in your app.

Install the package (if you didn't install it in Step 3):

```bash
npm install next-cloudinary
```

Add your Cloudinary cloud name to your environment variables in `.env.local` (or `.env`):

```bash
NEXT_PUBLIC_CLOUDINARY_CLOUD_NAME="<Your Cloud Name>"
```

Don't have a Cloudinary account? Sign up for free at `[https://cloudinary.com](https://cloudinary.com)`.

## Method 2: Add Components One by One

If you only need specific components, you can copy them individually.

### Step 1: Create Component File

Create a file for your component, for example:

```bash
mkdir -p components/ui
touch components/ui/button.tsx
```

### Step 2: Copy Component Code

Visit the component page on our documentation and click "Copy Code" to get the source.

### Step 3: Install Required Dependencies

Each component page shows which dependencies you need. For a button, you might need:

```bash
npm install @radix-ui/react-slot class-variance-authority clsx tailwind-merge
```

### Step 4: Use the Component

```tsx
import { Button } from "@/components/ui/button";

export default function Home() {
  return (
    <div>
      <Button>Click me</Button>
    </div>
  );
}
```

## Using Motion Components

Motion components add smooth animations to your app.

### Step 1: Install Motion

```bash
npm install motion
```

### Step 2: Copy Motion Component

Create a motion component file and copy the code from our documentation.

### Step 3: Use in Your App

```tsx
import { FadeIn } from "@/components/motion/fade-in";

export default function Home() {
  return (
    <FadeIn>
      <h1>Animated content</h1>
    </FadeIn>
  );
}
```

## Troubleshooting

### Common Issues

**Components not rendering?**

- Check that all dependencies are installed
- Verify your Tailwind CSS is properly configured
- Ensure the component file paths are correct

**Styles look wrong?**

- Make sure the CSS is added to your `globals.css`
- Check that the `ThemeProvider` is set up in your layout

**TypeScript errors?**

- Ensure you're using TypeScript in your project
- Check that all `@radix-ui` types are installed
