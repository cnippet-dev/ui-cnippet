## Get Started in 4 Simple Steps

<Steps>
<Step>
### Get the Code
Start by cloning our repository to get all components at once.

```bash
git clone https://github.com/cnippet-dev/all-elements
```

This gives you access to everything in one place.
</Step>

<Step>
### Set Up Your Project
Copy the components into your Next.js app and install dependencies.

```bash
# Copy these folders
components/
hooks/
lib/

# Install dependencies (see full list in Installation)
npm install @radix-ui/react-* motion recharts
```

[View full installation guide â†’](/docs/installation)
</Step>

<Step>
### Explore Components
Browse our component library to find what you need.

- **UI Components** - [/components](https://ui.cnippet.dev/components)
- **Animations** - [/motions](https://ui.cnippet.dev/motions)
- **Charts** - [/charts](https://ui.cnippet.dev/charts)

Each component includes live demos and copy-paste code.
</Step>

<Step>
### Build & Customize
Copy components directly into your project and customize them to match your brand.

```tsx
import { Button } from "@/components/ui/button";

export default function Home() {
  return <Button>Get Started</Button>;
}
```
</Step>
</Steps>
