## Usage

```tsx
import {
  Field,
  FieldError,
  FieldLabel,
} from "@/components/ui/field"
import { Form } from "@/components/ui/form"
import { Input } from "@/components/ui/input"
```

```tsx
<Form
  onSubmit={(e) => {
    /* handle submit */
  }}
>
  <Field>
    <FieldLabel>Email</FieldLabel>
    <Input name="email" type="email" required />
    <FieldError>Please enter a valid email.</FieldError>
  </Field>
</Form>
```

## API Reference

### Form

Styled wrapper around Base UI's `Form` primitive. Renders a `<form>` element and provides shared validation context to all `Field` children.

Accepts all standard `<form>` HTML attributes. Key props:

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `onSubmit` | `React.FormEventHandler<HTMLFormElement>` | — | Called when the form is submitted (after validation passes) |
| `errors` | `Record<string, string>` | — | Server-side error map keyed by field name; overrides client validation |
