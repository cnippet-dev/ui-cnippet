  </Label>

## API Reference

### CheckboxGroup

Provides shared state to a set of `Checkbox` components. Thin wrapper around Base UI's `CheckboxGroup`.

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `value` | `string[]` | — | Controlled array of checked values |
| `defaultValue` | `string[]` | `[]` | Initial checked values for uncontrolled usage |
| `onValueChange` | `(value: string[]) => void` | — | Callback when the selection changes |
| `allValues` | `string[]` | — | Full list of possible values; enables parent checkbox indeterminate state |
| `disabled` | `boolean` | `false` | Disables all checkboxes in the group |
