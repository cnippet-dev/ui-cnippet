## Usage

```tsx
import { Label } from "@/components/ui/label"
```

```tsx
<Label htmlFor="email">Email</Label>
```


## API Reference

### Label

Styled `<label>` element. Accepts all standard `<label>` HTML attributes.

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `htmlFor` | `string` | — | ID of the control this label is associated with |
| `render` | `React.ReactElement` | — | Renders a different element in place of `<label>` |
