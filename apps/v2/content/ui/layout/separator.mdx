## API Reference

### Separator

An accessible divider element.

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `orientation` | `"horizontal" \| "vertical"` | `"horizontal"` | Direction of the separator |

Renders a `<div>` with `role="separator"` (or `role="presentation"` when `decorative` is set). All standard `<div>` HTML attributes are forwarded.
