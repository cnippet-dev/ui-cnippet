## Usage

```tsx
import {
  Progress,
  ProgressLabel,
  ProgressValue,
} from "@/components/ui/progress"
```

```tsx
<Progress value={40} />
```

Note: If you render children inside `Progress`, you must also include `ProgressTrack` and `ProgressIndicator` inside it. Without them, the bar will not display. When no children are provided, a default track and indicator are rendered for you.

## API Reference

### Progress

Root container. Renders a default track and indicator when no children are provided.

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `value` | `number \| null` | `null` | Current progress value; `null` renders an indeterminate state |
| `max` | `number` | `100` | Maximum value |
| `min` | `number` | `0` | Minimum value |

### ProgressLabel

`<label>` element associated with the progress bar for screen readers.

### ProgressTrack

The background bar track. Rendered automatically inside `Progress` when no children are provided.

### ProgressIndicator

The filled portion of the bar. Width is driven by the `value`/`max`/`min` of the root.

### ProgressValue

Displays the current value as formatted text (e.g. `"40%"`).
