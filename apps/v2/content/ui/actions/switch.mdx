## API Reference

### Switch

Styled wrapper around Base UI's `Switch.Root` primitive.

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `size` | `"default" \| "sm"` | `"default"` | Size of the switch track and thumb |
| `checked` | `boolean` | — | Controlled checked state |
| `defaultChecked` | `boolean` | `false` | Initial checked state for uncontrolled usage |
| `onCheckedChange` | `(checked: boolean) => void` | — | Callback when checked state changes |
| `disabled` | `boolean` | `false` | Prevents interaction |
| `name` | `string` | — | Name used when submitting an HTML form |
| `required` | `boolean` | `false` | Whether the field must be checked before form submission |
| `readOnly` | `boolean` | `false` | Prevents the user from changing the checked state |
