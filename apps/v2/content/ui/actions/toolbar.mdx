## Usage

```tsx
import { Button } from "@/registry/default/ui/button"
import { Toggle } from "@/registry/default/ui/toggle"
import {
  Toolbar,
  ToolbarButton,
  ToolbarGroup,
  ToolbarSeparator,
} from "@/registry/default/ui/toolbar"
```

```tsx
<Toolbar>
  <ToolbarGroup>
    <ToolbarButton render={<Toggle />}>Bold</ToolbarButton>
    <ToolbarButton render={<Toggle />}>Underline</ToolbarButton>
  </ToolbarGroup>
  <ToolbarSeparator />
  <ToolbarButton render={<Button />}>Save</ToolbarButton>
</Toolbar>
```

## API Reference

All components are styled wrappers over Base UI's Toolbar primitives and forward all their respective props.

### Toolbar

Root container with role `toolbar`.

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `orientation` | `"horizontal" \| "vertical"` | `"horizontal"` | Layout direction; sets `aria-orientation` |

### ToolbarGroup

Groups related controls with a shared label for screen readers.

### ToolbarButton

An interactive button inside the toolbar. Use the `render` prop with `<Button>` or `<Toggle>` for full styling.

### ToolbarSeparator

Visual and semantic divider between toolbar sections.

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `orientation` | `"horizontal" \| "vertical"` | `"vertical"` | Direction of the separator line |

### ToolbarLink

An anchor element inside the toolbar that participates in toolbar keyboard navigation.

### ToolbarInput

An input element inside the toolbar that participates in toolbar keyboard navigation.

A full-width search input with a leading search icon via `ToolbarInput`, icon ghost buttons for filter, columns, export, and delete, and a primary "Add row" button on the right end.

A language `Select` dropdown, a word-wrap `ToggleGroupItem`, a copy icon button that switches to a check on success, and a loading-aware "Run" button — the essential code snippet toolbar.
