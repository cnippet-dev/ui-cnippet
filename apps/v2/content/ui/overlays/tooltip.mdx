## Usage

```tsx
import {
  Tooltip,
  TooltipContent,
  TooltipProvider,
  TooltipTrigger,
} from "@/components/ui/tooltip"
```

```tsx
<Tooltip>
  <TooltipTrigger render={<Button variant="outline" />}>
    Hover me
  </TooltipTrigger>
  <TooltipContent>Helpful hint</TooltipContent>
</Tooltip>
```

**Grouped Tooltips**

Wrap multiple tooltips in `TooltipProvider` so that once one becomes visible, adjacent tooltips appear instantly without the usual delay:

```tsx
<TooltipProvider>
  <Tooltip>
    <TooltipTrigger>...</TooltipTrigger>
    <TooltipContent>First</TooltipContent>
  </Tooltip>
  <Tooltip>
    <TooltipTrigger>...</TooltipTrigger>
    <TooltipContent>Second</TooltipContent>
  </Tooltip>
</TooltipProvider>
```

**TooltipContent Props**

- `side`: Which side the tooltip opens from. Options: `"top"` (default), `"bottom"`, `"left"`, `"right"`, `"inline-start"`, `"inline-end"`
- `align`: Alignment relative to the trigger. Options: `"center"` (default), `"start"`, `"end"`
- `sideOffset`: Gap in pixels between the trigger and tooltip. Default: `4`

```tsx
<TooltipContent side="bottom" align="start" sideOffset={8}>
  Opens below, aligned to the start
</TooltipContent>
```

## API Reference

### TooltipProvider

Wrap multiple tooltips in `TooltipProvider` so they share the same open delay. Once one tooltip opens, the rest appear instantly.

### Tooltip

Root provider for a single tooltip. Wraps Base UI `Tooltip.Root`.

### TooltipTrigger

The element that triggers the tooltip on hover or focus.

### TooltipPopup / TooltipContent

The floating popup label.

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `side` | `"top" \| "bottom" \| "left" \| "right" \| "inline-start" \| "inline-end"` | `"top"` | Side to open relative to the trigger |
| `align` | `"start" \| "center" \| "end"` | `"center"` | Alignment relative to the trigger |
| `sideOffset` | `number` | `4` | Gap in pixels from the trigger |
| `anchor` | `Element \| VirtualElement` | — | Custom anchor element for positioning |
| `portalProps` | `Portal.Props` | — | Props forwarded to the portal container |

### Animated Tooltips

Create tooltips that smoothly transition between triggers using a shared handle. All triggers share a single popup — position, size, and content animate automatically as focus moves between them.

To create detached triggers:

1. Create a handle using `TooltipCreateHandle`
2. Attach the same handle to multiple `TooltipTrigger` components
3. Each trigger provides a `payload` prop containing the content component
4. Use a single `Tooltip` component with the handle to render the popup
