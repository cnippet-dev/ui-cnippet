## Usage

```tsx
import {
  ScrollVelocityContainer,
  ScrollVelocityRow,
} from "@/components/motion/scroll-velocity-text";
```

```tsx
<ScrollVelocityContainer>
  <ScrollVelocityRow baseVelocity={3}>
    <span className="mx-6">React</span>
    <span className="mx-6">TypeScript</span>
    <span className="mx-6">Next.js</span>
  </ScrollVelocityRow>
</ScrollVelocityContainer>
```

## Props

### ScrollVelocityContainer

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `children` | `ReactNode` | — | One or more `ScrollVelocityRow` elements |
| `className` | `string` | — | Class applied to the outer wrapper |

### ScrollVelocityRow

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `children` | `ReactNode` | — | Content block that is repeated to fill the row |
| `baseVelocity` | `number` | `5` | Base scroll speed as a percentage of block width per second |
| `direction` | `1 \| -1` | `1` | Scroll direction — `1` for left-to-right, `-1` for right-to-left |
| `scrollReactivity` | `boolean` | `true` | When `true`, scroll velocity modifies the speed |
| `className` | `string` | — | Class applied to the row container |
