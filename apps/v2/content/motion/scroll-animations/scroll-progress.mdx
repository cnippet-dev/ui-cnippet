
```tsx
<ScrollProgress className="fixed inset-x-0 top-0 h-1 bg-primary" />
```

## Props

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `className` | `string` | — | Class applied to the progress bar element — use this to set colour, height, and position |
| `springOptions` | `SpringOptions` | `{ stiffness: 200, damping: 50, restDelta: 0.001 }` | Spring physics for smoothing the progress value |
| `containerRef` | `RefObject<HTMLDivElement>` | — | When provided, tracks scroll inside that container instead of the window |
