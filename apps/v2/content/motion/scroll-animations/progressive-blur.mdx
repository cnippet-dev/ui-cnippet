## Usage

```tsx
import { ProgressiveBlur } from "@/components/motion/progressive-blur";
```

```tsx
<div className="relative overflow-hidden">
  <YourContent />
  <ProgressiveBlur
    className="absolute inset-x-0 bottom-0 h-24"
    direction="bottom"
    blurLayers={6}
    blurIntensity={0.3}
  />
</div>
```

## Props

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `direction` | `"top" \| "bottom" \| "left" \| "right"` | `"bottom"` | Edge from which the blur fades in |
| `blurLayers` | `number` | `8` | Number of stacked blur layers — more layers = smoother gradient |
| `blurIntensity` | `number` | `0.25` | Blur amount per layer in pixels — total blur = `blurLayers × blurIntensity` |
| `className` | `string` | — | Size and position of the blur overlay |
