## Usage

```tsx
import {
  StackingCardItem,
  StackingCards,
} from "@/components/motion/stacking-cards";
```

```tsx
<StackingCards className="h-[300vh]" totalCards={3}>
  {cards.map((card, i) => (
    <StackingCardItem index={i} key={card.id} topPosition={`${5 + i * 5}%`}>
      <div className="h-64 rounded-2xl border bg-card p-6">
        {card.content}
      </div>
    </StackingCardItem>
  ))}
</StackingCards>
```

## Props

### StackingCards

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `totalCards` | `number` | — | Total number of card items — required for scale calculations |
| `scaleMultiplier` | `number` | `0.03` | How much each card shrinks relative to the one above it |
| `scrollOptions` | `UseScrollOptions` | — | Options passed to Motion's `useScroll` — use `container` to scope scroll to a div |
| `className` | `string` | — | Class applied to the outer container — set the height here |

### StackingCardItem

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `index` | `number` | — | Zero-based position of this card in the stack |
| `topPosition` | `string` | `"5 + index * 3%"` | CSS `top` value for the stacked offset (e.g. `"8%"`, `"24px"`) |
| `className` | `string` | — | Class applied to the sticky wrapper |
