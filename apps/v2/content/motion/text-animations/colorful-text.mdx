## Props

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `children` | `string` | — | Text to animate |
| `as` | `React.ElementType` | `"p"` | HTML element rendered as the container |
| `colors` | `string[]` | 10 vibrant defaults | Array of hex/CSS colours assigned randomly to characters |
| `trigger` | `boolean` | `true` | Controls animate/exit — set `false` to hide, `true` to play |
| `staggerDuration` | `number` | `0.04` | Seconds between each character |
| `transition` | `Transition` | spring | Per-character Motion transition |
| `className` | `string` | — | Class on the container element |
