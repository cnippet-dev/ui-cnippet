## Usage

```tsx

```

```tsx
<CountingNumber target={1000} />
```

## Props

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `target` | `number` | — | The final value to count to |
| `from` | `number` | `0` | Starting value |
| `autoStart` | `boolean` | `true` | Start counting immediately on mount |
| `transition` | `ValueAnimationTransition` | `{ type: "tween", duration: 3, ease: "easeInOut" }` | Motion transition controlling the count animation |
| `onStart` | `() => void` | — | Callback fired when counting starts |
| `onComplete` | `() => void` | — | Callback fired when counting reaches `target` |
| `className` | `string` | — | Class applied to the `<span>` element |

### Ref API

Pass a `ref` to access `startAnimation()` imperatively:

```tsx
const ref = useRef<CountingNumberRef>(null);
ref.current?.startAnimation(); // resets to `from` and counts again
```
