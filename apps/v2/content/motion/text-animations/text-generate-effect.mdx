## Props

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `children` | `string` | — | Text to generate — split by whitespace into words |
| `as` | `React.ElementType` | `"p"` | HTML element rendered as the container |
| `filter` | `boolean` | `true` | Apply blur unblur effect to each word |
| `trigger` | `boolean` | `true` | Set `false` to hold hidden; `true` to play |
| `staggerDuration` | `number` | `0.1` | Seconds between each word |
| `transition` | `Transition` | `{ duration: 0.5 }` | Per-word Motion transition |
| `className` | `string` | — | Class on the container |
| `wordClassName` | `string` | — | Class applied to each word span |
