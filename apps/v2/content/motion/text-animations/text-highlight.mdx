## Usage

```tsx
import { TextHighlight } from "@/components/motion/text-highlight";
```

```tsx
<TextHighlight triggerType="inView" highlightColor="hsl(262, 80%, 85%)">
  production-ready
</TextHighlight>
```

## Props

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `children` | `ReactNode` | — | Text or content to highlight |
| `triggerType` | `"inView" \| "hover" \| "ref" \| "auto"` | `"inView"` | What triggers the highlight animation |
| `highlightColor` | `string` | `"hsl(25, 90%, 80%)"` | CSS colour value for the highlight fill |
| `direction` | `"ltr" \| "rtl" \| "ttb" \| "btt"` | `"ltr"` | Direction the highlight sweeps in from |
| `transition` | `Transition` | spring | Motion transition for the sweep |
| `useInViewOptions` | `UseInViewOptions` | `{ once: true, amount: 0.1 }` | Options passed to `useInView` when `triggerType="inView"` |
| `as` | `ElementType` | `"span"` | HTML element for the outer wrapper |
| `className` | `string` | — | Class applied to the animated highlight span |

### Ref API

With `triggerType="ref"`, control the animation imperatively:

```tsx
const ref = useRef<TextHighlightRef>(null);
ref.current?.animate("rtl"); // sweep in from right
ref.current?.reset();        // remove highlight
```
