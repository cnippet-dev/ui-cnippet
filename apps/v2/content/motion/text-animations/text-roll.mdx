## Props

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `children` | `string` | — | The text to animate letter by letter |
| `duration` | `number` | `0.5` | Duration of each letter's roll animation (seconds) |
| `getEnterDelay` | `(index: number) => number` | `(i) => i * 0.1` | Delay for the entering letter at position `i` |
| `getExitDelay` | `(index: number) => number` | `(i) => i * 0.1 + 0.2` | Delay for the exiting letter at position `i` |
| `transition` | `Transition` | `{ ease: "easeIn" }` | Motion transition shared by all letters |
| `variants` | `{ enter: { initial, animate }, exit: { initial, animate } }` | — | Override the enter/exit rotation variants |
| `onAnimationComplete` | `() => void` | — | Callback fired when the last letter finishes |
| `className` | `string` | — | Class applied to the wrapping `<span>` |
