## Props

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `children` | `string` | — | The final resolved text |
| `speed` | `number` | `50` | Milliseconds between each scramble tick |
| `maxIterations` | `number` | `15` | Number of scramble ticks before fully resolving |
| `revealDirection` | `"ltr" \| "rtl" \| "center" \| "random"` | `"ltr"` | Order in which characters lock to their final value |
| `trigger` | `boolean` | `true` | Set `false` to pause; set back to `true` to replay |
| `onComplete` | `() => void` | — | Callback fired when all characters have resolved |
| `className` | `string` | — | Class on the outer `<span>` — always renders in `font-mono` |
