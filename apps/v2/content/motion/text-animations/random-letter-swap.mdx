## Props

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `label` | `string` | — | The text to display and animate |
| `reverse` | `boolean` | `true` | `true` slides letters downward; `false` slides them upward |
| `staggerDuration` | `number` | `0.02` | Seconds between each letter in the random sequence |
| `transition` | `AnimationOptions` | `{ type: "spring", duration: 0.8 }` | Motion transition for each letter swap |
| `className` | `string` | — | Class on the outer `<motion.span>` |
| `onClick` | `() => void` | — | Click handler |

## Examples

### Navigation Links

Each link shuffles independently on hover. The random order makes each nav item feel distinct even though they share the same component.

### CTA Buttons

Two call-to-action buttons with `reverse={false}` on the primary so the letters slide up instead of down, creating visual contrast.

Technology tags inside pill badges shuffle on hover. Short words with fast `staggerDuration` produce a rapid glitch pop.
