```tsx

```

```tsx
<SlidingNumber value={count} />
```

## Props

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `value` | `number` | — | The current number to display — update this prop to trigger the slide animation |
| `padStart` | `boolean` | `false` | Pad single-digit values with a leading zero (e.g. `07`) |
| `decimalSeparator` | `string` | `"."` | Character used between integer and decimal parts |

## Examples

### Increment / Decrement

Increment and decrement buttons drive a single `SlidingNumber`. Each digit column slides independently so only the changed place animates.

<ComponentPreview name="m-sliding-number-1" />

### Live Clock

Hours, minutes, and seconds each rendered as a separate `SlidingNumber` with `padStart`. The seconds column slides every tick while higher columns stay still.

<ComponentPreview name="m-sliding-number-2" />

### Pricing Plan Switcher

Switching between plan tiers updates both price and seat count simultaneously. The odometer motion makes the price change feel tangible rather than instant.
