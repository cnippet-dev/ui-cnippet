## Usage

```tsx
import { TextGradient } from "@/components/motion/text-gradient";
```

```tsx
<TextGradient colors={["#a855f7", "#3b82f6", "#06b6d4", "#a855f7"]}>
  Build with motion
</TextGradient>
```

## Props

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `children` | `string` | — | Text to apply the gradient to |
| `as` | `React.ElementType` | `"p"` | HTML element rendered as the container |
| `colors` | `string[]` | `["#ff6b6b", "#ffd93d", "#6bcb77", "#4d96ff", "#c77dff"]` | Array of colour stops for the gradient |
| `duration` | `number` | `4` | Seconds for one full gradient cycle |
| `angle` | `number` | `135` | Gradient angle in degrees |
| `transition` | `Transition` | — | Override the Motion transition |
| `className` | `string` | — | Additional classes for font size, weight, etc. |
