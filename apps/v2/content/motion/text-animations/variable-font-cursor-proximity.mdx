## Usage

```tsx
import { VariableFontCursorProximity } from "@/components/motion/variable-font-cursor-proximity";
```

```tsx
const containerRef = useRef<HTMLDivElement>(null);

<div ref={containerRef}>
  <VariableFontCursorProximity
    containerRef={containerRef}
    fromFontVariationSettings="'wght' 100"
    toFontVariationSettings="'wght' 900"
    radius={80}
  >
    Hover me
  </VariableFontCursorProximity>
</div>
```

> **Variable font required.** This component animates `font-variation-settings` directly. Your project must load a variable font that supports the axes you specify (e.g. Inter Variable for `wght` and `slnt`).
