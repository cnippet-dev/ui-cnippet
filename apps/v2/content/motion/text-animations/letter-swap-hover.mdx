## Props

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `label` | `string` | — | The text to display and animate |
| `reverse` | `boolean` | `true` | `true` slides letters down; `false` slides them up |
| `staggerDuration` | `number` | `0.03` | Seconds between each letter's animation |
| `staggerFrom` | `"first" \| "last" \| "center" \| number` | `"first"` | Which end the stagger wave originates from |
| `transition` | `AnimationOptions` | `{ type: "spring", duration: 0.7 }` | Motion transition for the swap |
| `className` | `string` | — | Class applied to the outer span |
| `onClick` | `() => void` | — | Click handler |
