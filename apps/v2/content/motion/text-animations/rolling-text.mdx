## Props

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `children` | `string` | — | Text to animate |
| `direction` | `"up" \| "down"` | `"up"` | `"up"` rolls in from below; `"down"` rolls in from above |
| `duration` | `number` | `0.6` | Seconds per letter's animation |
| `delay` | `number` | `0` | Initial delay before the first letter starts |
| `staggerDuration` | `number` | `0.04` | Seconds between each letter |
| `perspective` | `number` | `400` | CSS perspective depth in px |
| `transition` | `Transition` | cubic-bezier easing | Per-letter transition override |
| `className` | `string` | — | Class on the outer `<span>` |
