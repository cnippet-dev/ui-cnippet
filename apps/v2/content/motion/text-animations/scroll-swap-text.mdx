## Usage

```tsx
import { ScrollSwapText } from "@/components/motion/scroll-swap-text";
```

```tsx
const containerRef = useRef<HTMLDivElement>(null);

<div ref={containerRef} className="h-64 overflow-y-scroll">
  <ScrollSwapText containerRef={containerRef}>
    Scrollable text item
  </ScrollSwapText>
</div>
```

## Props

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `children` | `React.ReactNode` | — | Text content to display and animate |
| `as` | `ElementType` | `"span"` | HTML element rendered as the container |
| `containerRef` | `RefObject<HTMLElement>` | — | **Required.** Ref to the scrollable container that drives the animation |
| `offset` | `[string, string]` | `["0 0", "0 1"]` | Scroll offset range passed to Motion's `useScroll` |
| `springConfig` | `{ stiffness?, damping?, mass? }` | `{ stiffness: 200, damping: 30 }` | Spring configuration for the scroll animation |
| `className` | `string` | — | Class on the outer element |
