## Usage

```tsx
import { BreathingText } from "@/components/motion/breathing-text";
```

```tsx
<BreathingText
  fromFontVariationSettings="'wght' 300, 'wdth' 75"
  toFontVariationSettings="'wght' 900, 'wdth' 125"
>
  Build with motion
</BreathingText>
```

> **Variable font required** — `BreathingText` animates `font-variation-settings`. The effect is only visible when the active font supports the axes used (e.g. `wght`, `wdth`). Inter Variable, Geist, and most Google Fonts variable fonts work out of the box.

## Props

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `children` | `ReactNode` | — | Text content to animate |
| `fromFontVariationSettings` | `string` | — | Starting CSS `font-variation-settings` value |
| `toFontVariationSettings` | `string` | — | Peak CSS `font-variation-settings` value |
| `transition` | `Transition` | `{ duration: 1.5, ease: "easeInOut" }` | Transition for the breath cycle |
| `staggerDuration` | `number` | `0.1` | Seconds between each letter's cycle offset |
| `staggerFrom` | `"first" \| "last" \| "center" \| number` | `"first"` | Wave origin |
| `repeatDelay` | `number` | `0.1` | Pause between each repeat cycle |
| `as` | `ElementType` | `"span"` | HTML element rendered as the container |
| `className` | `string` | — | Class applied to the container |
