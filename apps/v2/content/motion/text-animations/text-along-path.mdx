## Usage

```tsx
import { TextAlongPath } from "@/components/motion/text-along-path";
```

```tsx
<TextAlongPath
  path="M 50,10 A 40,40 0 1,1 49.99,10"
  text="Circular text • "
  viewBox="0 0 100 100"
  animationType="auto"
  duration={6}
/>
```

## Props

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `path` | `string` | — | **Required.** SVG path `d` attribute — any valid path string |
| `text` | `string` | — | **Required.** Text content placed along the path |
| `animationType` | `"auto" \| "scroll"` | `"auto"` | `"auto"` loops continuously; `"scroll"` ties offset to scroll position |
| `duration` | `number` | `4` | Seconds for one full loop (auto mode only) |
| `repeatCount` | `number \| "indefinite"` | `"indefinite"` | Number of animation repetitions (auto mode only) |
| `viewBox` | `string` | `"0 0 100 100"` | SVG viewBox — coordinate space for the path |
| `width` | `string \| number` | `"100%"` | SVG width |
| `height` | `string \| number` | `"100%"` | SVG height |
| `showPath` | `boolean` | `false` | Render the path stroke for visual debugging |
| `pathClassName` | `string` | — | Class on the `<path>` element |
| `textClassName` | `string` | — | Class on the `<textPath>` element |
| `textAnchor` | `"start" \| "middle" \| "end"` | `"start"` | SVG text anchor |
| `svgClassName` | `string` | — | Class on the outer `<svg>` |
| `scrollContainer` | `RefObject<HTMLElement>` | — | Scrollable container for scroll mode |
| `scrollOffset` | `UseScrollOptions["offset"]` | `["start end", "end end"]` | Scroll offset range for scroll mode |
| `scrollTransformValues` | `[number, number]` | `[0, 100]` | Path offset range (%) mapped to scroll progress |

Text loops continuously around a circular path — classic badge or stamp design. Repeat the text with a bullet separator so the loop is seamless.
