## Usage

```tsx
import { SplittingText } from "@/components/motion/splitting-text";
```

```tsx
<SplittingText preset="slide-up" splitBy="words">
  Motion components for modern web apps
</SplittingText>
```

## Props

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `children` | `string` | — | Text to split and animate |
| `splitBy` | `"words" \| "chars" \| "lines"` | `"words"` | Granularity of the split |
| `preset` | `"fade" \| "slide-up" \| "slide-down" \| "blur" \| "scale"` | `"slide-up"` | Built-in animation style |
| `staggerDuration` | `number` | `0.05` | Seconds between each segment |
| `trigger` | `boolean` | `true` | Set `false` to hold hidden; set back to `true` to play |
| `transition` | `Transition` | spring | Per-segment transition |
| `as` | `ElementType` | `"p"` | HTML element for the container |
| `className` | `string` | — | Class on the container |
| `segmentClassName` | `string` | — | Class on each animated segment span |
