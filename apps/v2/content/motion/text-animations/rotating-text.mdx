```tsx

```

```tsx
<RotatingText words={["beautiful", "accessible", "composable"]} />
```

## Props

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `words` | `string[]` | — | Array of strings to rotate through |
| `interval` | `number` | `2500` | Milliseconds each word is shown |
| `direction` | `"up" \| "down"` | `"up"` | `"up"` enters from below; `"down"` enters from above |
| `transition` | `Transition` | spring | Motion transition for enter/exit |
| `className` | `string` | — | Class on the outer wrapper — use for font styling and colour |
