
## Props

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `texts` | `string[]` | — | Array of strings to cycle through |
| `rotationInterval` | `number` | `2000` | Milliseconds between automatic advances |
| `auto` | `boolean` | `true` | Advance automatically on an interval |
| `loop` | `boolean` | `true` | Wrap back to the first string after the last |
| `splitBy` | `"characters" \| "words" \| "lines" \| string` | `"characters"` | How to split each string for staggering |
| `staggerDuration` | `number` | `0` | Seconds between each segment's animation |
| `staggerFrom` | `"first" \| "last" \| "center" \| "random" \| number` | `"first"` | Which end the stagger starts from |
| `initial` | `MotionProps["initial"]` | `{ opacity: 0, y: "100%" }` | Enter-from state |
| `animate` | `MotionProps["animate"]` | `{ opacity: 1, y: 0 }` | Visible state |
| `exit` | `MotionProps["exit"]` | `{ opacity: 0, y: "-120%" }` | Exit-to state |
| `transition` | `Transition` | spring | Shared transition for all segments |
| `animatePresenceMode` | `AnimatePresenceProps["mode"]` | `"wait"` | AnimatePresence mode |
| `as` | `ElementType` | `"p"` | HTML element rendered as the container |
| `mainClassName` | `string` | — | Class on the outer container |
| `splitLevelClassName` | `string` | — | Class on each word wrapper |
| `elementLevelClassName` | `string` | — | Class on each character wrapper |
| `onNext` | `(index: number) => void` | — | Callback fired on each advance |

### Ref API

Pass a `ref` to control rotation imperatively:

```tsx
const ref = useRef<TextRotateRef>(null);
ref.current?.next();      // advance forward
ref.current?.previous();  // go back
ref.current?.jumpTo(2);   // jump to index
ref.current?.reset();     // go to index 0
```
