## Props

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `value` | `number` | — | Target value — update to trigger animation |
| `springOptions` | `SpringOptions` | — | Motion spring config (`stiffness`, `damping`, `mass`) |
| `as` | `React.ElementType` | `"span"` | HTML element rendered as the container |
| `className` | `string` | — | Class for font styling |

## Examples

### Stepper

+10 / −10 buttons update the value. The spring eases through intermediate numbers, making large jumps feel smooth rather than jarring.

<ComponentPreview name="m-animated-number-1" />

### Upload Progress

A percentage figure animates alongside a CSS progress bar. Both update from the same state, keeping the animated label in sync with the visual indicator.

<ComponentPreview name="m-animated-number-2" />

### Dashboard Metrics

Three KPI cards share a multiplier toggle (1×, 2×, 5×). All three counters spring to their new values simultaneously, giving the dashboard a lively feel on range changes.
