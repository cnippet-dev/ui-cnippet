## Props

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `children` | `string` | — | The string to display; change this prop to trigger morphing |
| `as` | `React.ElementType` | `"p"` | HTML tag for the container |
| `transition` | `Transition` | spring (stiffness 280) | Motion transition for character movement |
| `variants` | `Variants` | opacity fade | Custom enter/exit variants per character |
| `className` | `string` | — | Class applied to the container |
| `style` | `React.CSSProperties` | — | Inline style for the container |

## Examples

### Word Cycler

A button group selects the active word and passes it as `children`. Characters shared between words animate smoothly to their new positions.

<ComponentPreview name="m-text-morph-1" />

### Auto-Cycling Plan Name

An interval drives the `children` prop through an array. The tighter spring `transition` creates a snappier feel suitable for compact UI.

<ComponentPreview name="m-text-morph-2" />

### Morphing Placeholder

The placeholder string cycles as the user types, morphing character-by-character. Controlled by `query.length % suggestions.length` so each keystroke advances the suggestion.
