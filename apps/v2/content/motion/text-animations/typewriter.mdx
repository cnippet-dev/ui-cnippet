## Props

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `text` | `string \| string[]` | — | Text or array of texts to type through |
| `speed` | `number` | `50` | Typing speed in ms per character |
| `deleteSpeed` | `number` | `30` | Deletion speed in ms per character |
| `initialDelay` | `number` | `0` | Delay in ms before typing starts |
| `waitTime` | `number` | `2000` | Pause in ms before deleting (multi-string) |
| `loop` | `boolean` | `true` | Whether to loop through strings indefinitely |
| `showCursor` | `boolean` | `true` | Show the blinking cursor |
| `hideCursorOnType` | `boolean` | `false` | Hide cursor while actively typing |
| `cursorChar` | `string \| ReactNode` | `"\|"` | Character or element for the cursor |
| `cursorClassName` | `string` | `"ml-1"` | Class applied to the cursor element |
| `as` | `React.ElementType` | `"div"` | HTML tag for the container |
| `className` | `string` | — | Class applied to the container |
