## Usage

```tsx
import { TextCursorProximity } from "@/components/motion/text-cursor-proximity";
```

```tsx
const containerRef = useRef<HTMLDivElement>(null);

<div ref={containerRef}>
  <TextCursorProximity
    containerRef={containerRef}
    styles={{ scale: { from: 1, to: 1.5 } }}
    radius={60}
  >
    Hover over me
  </TextCursorProximity>
</div>
```

## Props

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `children` | `React.ReactNode` | — | Text to animate |
| `as` | `ElementType` | `"span"` | HTML element rendered as the container |
| `styles` | `Partial<Record<CSSProperty, { from, to }>>` | — | **Required.** Map of CSS properties with `from` and `to` values driven by cursor proximity |
| `containerRef` | `RefObject<HTMLDivElement>` | — | **Required.** Ref of the element used as the mouse coordinate space |
| `radius` | `number` | `50` | Pixel radius within which the effect is active |
| `falloff` | `"linear" \| "exponential" \| "gaussian"` | `"linear"` | How the effect attenuates with distance |
| `className` | `string` | — | Class on the outer element |

## Examples

### Scale

Characters scale up as the cursor approaches. The `radius` controls how large the influence zone is.
