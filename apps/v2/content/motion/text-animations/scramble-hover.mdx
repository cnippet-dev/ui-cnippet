
```tsx

```

## Props

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `children` | `string` | — | The text to scramble on hover |
| `scrambleSpeed` | `number` | `50` | Interval in ms between scramble ticks |
| `maxIterations` | `number` | `10` | Number of scramble steps before resolving |
| `revealDirection` | `"start" \| "end" \| "center" \| "random"` | `"start"` | Direction characters resolve from |
| `useOriginalCharsOnly` | `boolean` | `false` | Restrict scramble pool to original characters |
| `characterSet` | `string` | A–Z, a–z, 0–9, symbols | Characters to scramble through |
| `onHoverStart` | `() => void` | — | Callback fired when hover starts |
| `onHoverEnd` | `() => void` | — | Callback fired when hover ends |
| `className` | `string` | — | Class applied to the wrapper span |
