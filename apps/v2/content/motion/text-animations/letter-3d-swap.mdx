## Usage

```tsx
import { Letter3DSwap } from "@/components/motion/letter-3d-swap";
```

```tsx
<Letter3DSwap rotateDirection="top" frontFaceClassName="text-foreground" secondFaceClassName="text-primary">
  Get started
</Letter3DSwap>
```

## Props

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `children` | `string \| number` | — | Text to animate |
| `as` | `ElementType` | `"p"` | HTML element rendered as the container |
| `rotateDirection` | `"top" \| "right" \| "bottom" \| "left"` | `"right"` | Which axis the letters rotate around |
| `staggerDuration` | `number` | `0.05` | Seconds between each character's rotation |
| `staggerFrom` | `"first" \| "last" \| "center" \| "random" \| number` | `"first"` | Stagger wave origin |
| `transition` | `AnimationOptions` | spring | Motion transition for the rotation |
| `frontFaceClassName` | `string` | — | Class on the visible (front) face |
| `secondFaceClassName` | `string` | — | Class on the hidden (back) face shown during hover |
| `mainClassName` | `string` | — | Class on the outer element |

## Examples

### Navigation Links

Nav links rotate upward on hover (`rotateDirection="top"`). Front face uses `text-muted-foreground`, second face reveals `text-foreground` — a colour-change through 3D space.

<ComponentPreview name="m-letter-3d-swap-1" />

### CTA Buttons

Two buttons — filled and outlined — with opposite `rotateDirection` values and `staggerFrom="last"` on the secondary to reverse the wave.

<ComponentPreview name="m-letter-3d-swap-2" />

### Direction Showcase

Four rows of the same word, each with a different `rotateDirection`. Hover to compare `top`, `right`, `bottom`, and `left` rotations side by side.
