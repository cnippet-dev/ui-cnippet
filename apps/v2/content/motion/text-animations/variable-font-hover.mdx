## Usage

```tsx
import { VariableFontHover } from "@/components/motion/variable-font-hover";
```

```tsx
<VariableFontHover
  label="Get started"
  fromFontVariationSettings="'wght' 400"
  toFontVariationSettings="'wght' 800"
/>
```

> **Variable font required.** This component animates `font-variation-settings` — your project must load a variable font that supports the axes you specify (e.g. Inter Variable for `wght` and `slnt`).

## Props

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `label` | `string` | — | The text to display and animate |
| `fromFontVariationSettings` | `string` | — | Font variation settings at rest (e.g. `"'wght' 400"`) |
| `toFontVariationSettings` | `string` | — | Font variation settings on hover (e.g. `"'wght' 800"`) |
| `staggerDuration` | `number` | `0.03` | Seconds between each letter's transition |
| `staggerFrom` | `"first" \| "last" \| "center" \| number` | `"first"` | Where the stagger wave starts |
| `transition` | `AnimationOptions` | `{ type: "spring", duration: 0.7 }` | Motion transition applied to each letter |
| `className` | `string` | — | Class on the outer `<motion.span>` |
| `onClick` | `() => void` | — | Click handler |

## Examples

### Navigation Links

Nav links bold from the center outward on hover. `staggerFrom="center"` creates a symmetric wave that feels more natural for short words.

### Hero Headline

Two headline lines with inverted settings — the first goes thin-to-bold while the second goes bold-to-thin, creating contrast as you hover between them.

### CTA Links

Three call-to-action links each use a different `staggerFrom` origin — `first`, `center`, and `last` — so each line feels distinct despite sharing the same component.
