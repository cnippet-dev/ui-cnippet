## Usage

```tsx
import { FlipWords } from "@/components/motion/flip-words";
```

```tsx
<FlipWords words={["beautiful", "accessible", "composable"]} />
```

## Props

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `words` | `string[]` | — | Array of strings to cycle through |
| `duration` | `number` | `3000` | Milliseconds each word is displayed before flipping |
| `transition` | `Transition` | `{ type: "spring", stiffness: 100, damping: 14 }` | Motion transition for the flip animation |
| `className` | `string` | — | Class applied to the outer wrapper span |

## Examples

### Hero Inline

Words flip inline inside a heading. Use `className` to colour the cycling word differently from the surrounding text.
