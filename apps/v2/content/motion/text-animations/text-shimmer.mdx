## Props

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `children` | `string` | — | Text to apply the shimmer to |
| `as` | `React.ElementType` | `"p"` | HTML element rendered as the container |
| `duration` | `number` | `2` | Seconds for one full shimmer pass |
| `spread` | `number` | `2` | Multiplied by character count to determine highlight width in px |
| `className` | `string` | — | Additional classes — use to set font size, weight, and custom color tokens |
