## Usage

```tsx
import { MorphingText } from "@/components/motion/morphing-text";
```

```tsx
<MorphingText texts={["Innovate", "Create", "Inspire", "Deliver"]} />
```

## Props

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `texts` | `string[]` | — | Array of strings to morph through in sequence |
| `className` | `string` | — | Class on the container — use to set text colour. Font size is controlled by the built-in `[font-size:40pt]` default; override with a utility like `[font-size:32pt]` |

> **How it works** — The component uses a `requestAnimationFrame` loop that drives two overlapping `<span>` elements. A SVG `feColorMatrix` filter thresholds the combined blur, making partial overlap appear as a solid glyph rather than a blurred smear. The `morph` / `cooldown` timing constants are internal and not currently exposed as props.
