## Usage

```tsx
import { TextScramble } from "@/components/motion/text-scramble";
```

```tsx
<TextScramble duration={0.8}>Your text here</TextScramble>
```

## Props

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `children` | `string` | — | The target text to resolve to |
| `duration` | `number` | `0.8` | Total scramble duration in seconds |
| `speed` | `number` | `0.04` | Interval between scramble steps in seconds |
| `characterSet` | `string` | A–Z, a–z, 0–9 | Characters to scramble through |
| `trigger` | `boolean` | `true` | Set `false` to pause, `true` to play |
| `as` | `React.ElementType` | `"p"` | HTML tag for the container |
| `onScrambleComplete` | `() => void` | — | Callback fired when scramble finishes |
| `className` | `string` | — | Class applied to the container |
