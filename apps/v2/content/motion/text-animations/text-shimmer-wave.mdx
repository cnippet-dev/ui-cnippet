## Props

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `children` | `string` | — | Text to animate |
| `as` | `React.ElementType` | `"p"` | HTML element rendered as the container |
| `duration` | `number` | `1` | Seconds for the wave to travel the full string |
| `spread` | `number` | `1` | Higher values compress the wave delay, making it travel faster relative to string length |
| `zDistance` | `number` | `10` | Peak z-axis translation in px |
| `xDistance` | `number` | `2` | Peak x-axis translation in px |
| `yDistance` | `number` | `-2` | Peak y-axis translation in px (negative = up) |
| `scaleDistance` | `number` | `1.1` | Peak scale multiplier at the wave crest |
| `rotateYDistance` | `number` | `10` | Peak Y-axis rotation in degrees at the crest |
| `transition` | `Transition` | — | Override the per-character transition |
| `className` | `string` | — | Class on the container element |
