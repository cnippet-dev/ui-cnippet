# Cnippet UI Roadmap

## Now (v0.x)

- [x] 40+ core components
- [x] CLI installation
- [x] Dark mode support
- [x] TypeScript native
- [ ] Live demo site (play.cnippet.dev)
- [ ] Storybook integration

## Next (v0.x+1)

- [ ] Component themes (e.g., minimal, brutalist, glassmorphism)
- [ ] Motion/animation variants for all components
- [ ] Figma component kit
- [ ] VS Code extension for component insertion

## Later (v1.0)

- [ ] Right-to-left (RTL) support
- [ ] Framework adapters (Vite, Remix, Astro)
- [ ] Component blocks (full page sections)
- [ ] Sponsor tiers & open collective

## Community Requested

Open an issue with `[Feature]` in the title to suggest something, or vote on existing requests in [GitHub Discussions](https://github.com/cnippet-dev/ui-cnippet/discussions).
